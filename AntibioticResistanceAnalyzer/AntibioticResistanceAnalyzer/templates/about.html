About AMR Mutation Analyzer

Overview

The AMR Mutation Analyzer is a comprehensive bioinformatics tool designed to identify genetic mutations associated with antibiotic resistance in bacterial genomes. By comparing resistant and non-resistant strains, the tool identifies mutations that are statistically associated with resistance phenotypes.

Our application combines statistical analysis with machine learning approaches to provide a robust identification of resistance-linked mutations. The results are presented through intuitive visualizations and detailed reports that can be exported for further analysis.

Methodology

Our analysis pipeline includes:
  1. Data Preprocessing: Loading and processing genomic data from various formats (FASTA, CSV, TSV, JSON)
  2. Mutation Identification: Identifying genetic variations between resistant and non-resistant strains
  3. Statistical Analysis: Group Association Model (GAM) for comparing mutation frequencies with p-value calculations
  4. Feature Selection: Using XGBoost to identify the most informative mutations
  5. Classification: Training SVM models to predict resistance status based on genetic features
  6. Model Validation: 10-fold cross-validation to ensure reliability
  7. Structural Mapping: Mapping significant mutations to 3D protein structures

Using the Application

Getting Started:
  1. Create an account to manage your projects
  2. Create a new project for your analysis
  3. Upload genome data for resistant and non-resistant strains
  4. Run analysis with desired parameters
  5. Explore results through interactive visualizations
  6. Export findings for publication or further analysis
Supported Data Formats:
  • FASTA (.fa, .fasta, .fna): Genomic sequence data
  • CSV (.csv): Tabular data with sequences or features
  • TSV (.tsv): Tab-separated values with genomic data
  • JSON (.json): Structured genomic data from databases
